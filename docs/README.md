# Yvynation - Earth Engine App

A Python-based Earth Engine application for publishing geospatial analysis and visualization.

## Project Structure

- `main.py` - Main Earth Engine application entry point
- `config.py` - Configuration and constants
- `requirements.txt` - Python dependencies
- `data/` - Data files (GeoJSON, shapefiles, etc.)
- `scripts/` - Utility scripts for EE tasks

## Getting Started

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Earth Engine

```bash
earthengine authenticate
```

### Run the App

```bash
python main.py
```

## Earth Engine Repository

This repository is configured as an Earth Engine code repository:
- **Remote:** `https://earthengine.googlesource.com/users/leandromet/yvynation`
- **Access:** Via [Earth Engine Code Editor](https://code.earthengine.google.com/?accept_repo=users/leandromet/yvynation)
